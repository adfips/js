Что бы запустить программу на языке Rutra, в консоле нужно передать следующие аргументы: node <nameInterpreter>.js <nameCode>.rr arg1 arg2...

input arg1 | ввод.
output arg1 | ввыод.
set arg1 value | в ячейку памяти arg1 положить value
expres arg1 arg2 arg3 | arg1,arg2 - слогаемые, arg3 - их сумма.
diff arg1 arg2 arg3 | arg1,arg2 - уменьшаемое, вычетаемое соответственно, arg3 - их разность.
multi arg1 arg2 arg3 | arg1,arg2 - множители, arg3 - их произведение.
cmp arg1 >/</=/!= arg2 | сравнивает arg1 c arg2.
    Если true, то выполняет без "прыжков" в ином случае перепрыгнет через строку.
    Опреции сравнения: <,>,=,!= меньше, больше, равно, не равно соответственно.
jmps | нацало цикла.
jmpf | конец цикла (Пишется без условий. Для выхода из цикла следует написать условие над jmpf).
# - комментирование (После знака # для корректной работы нужно поставить пробел)
*arg1,arg2... - ячейки памяти
